# JUSTIFYING THE RETENTION OF CERTAIN LOCAL GREEN SPACES IN COULSDON

The official inspection of Croydon's new five year local plan demanded that a large number of very local green spaces in the borough need stronger justification if they are to be retained. (Behind this decision is pressure to find much more land in the borough on which to build dwellings "given planning constraints and development requirements".)
