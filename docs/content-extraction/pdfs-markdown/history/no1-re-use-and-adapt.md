## Design Concept

This scheme represents a "re-use and adapt" approach to the Farthing Down Kiosk development.

## Building Elements Shown

### Elevations
- **East Elevation** - New roof provides sheltered access to kiosk, with possible access to bin area
- **North Elevation** - Large fold back doors to kiosk to be used for interpretive panels, cut back roof extended to provide shelter and improve appearance of building
- **South Elevation** - Cut back roof extended to provide shelter and improve appearance of building

### Site Plan Features
- New roof to side access
- New fold-back doors access
- Access to building (internal and for servicing)
- Kiosk
- Servery
- Wc/lobby/changing facilities
- Store/storage area
- Hardware store
- Existing buildings (shown on plan)
- New access roof for rubbish and servicing/interpretation
- Lawn area
- Defined area for possible tables
- Recycling and refuse area
- Fenced
- Seating
