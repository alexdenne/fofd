# Farthing Down Kiosk - Modern Building Proposal

## Overview

This document presents architectural plans for a proposed modern building at Farthing Down, designated as "sketch scheme 3". The plans were created by SALT Architects of East Yorkshire, Hull, for client LUC (Land Use Consultants).

## Building Design

### Architectural Features

The proposed building features a distinctive **barrel-vaulted roof** with a modern design aesthetic. The structure is designed to blend with the natural surroundings of Farthing Down while providing essential visitor facilities.

### Elevations

The architectural drawings include:
- **Elevation to Car Park**: Shows the front-facing view of the building with the characteristic curved roof structure and entrance area
- Multiple perspective views demonstrating how the building integrates with the landscape

## Facility Layout

The building includes the following functional areas:

### Main Facilities
- **Food preparation** area
- **Food store**
- **Disabled WC** (wheelchair accessible toilet)
- **WC** (standard toilet facilities)
- **Interpretation boards** area for visitor information

### Site Features
- **Security skylight** for natural lighting and security
- **Grass banked steps** providing landscape integration
- Defined area for possible future kiosk expansion

## Site Location

The proposed building is situated near:
- **Sleeping Stone Cottage**
- **The Old Cottage**

The site plan shows careful consideration of the existing landscape and neighboring structures.
