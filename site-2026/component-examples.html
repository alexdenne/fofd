JavaScript Components Demo

Interactive examples of the FoFD website components. All components use vanilla JavaScript with no dependencies.

Audio Player Component

Nature Trail audio guide with multilingual support (EN/FR/HI/PL)

Features:

  • 29 stops with audio in 4 languages
  • Play/pause controls with progress bar
  • Previous/next navigation
  • Volume control
  • Time display
  • Keyboard shortcuts (Space to play/pause, Arrow keys for navigation)
  • Mobile responsive
  • Fully accessible with ARIA labels

Usage

<!-- Add to your HTML -->
<div id="nature-trail-player" data-audio-player></div>

<!-- Include CSS and JS -->
<link rel="stylesheet" href="css/style.css">
<script src="js/components.js"></script>

<!-- Components auto-initialize, or initialize manually: -->
<script>
  const player = new AudioPlayer('nature-trail-player', {
    audioBasePath: '/site-2005-2025/NatureTrail',
    totalStops: 29,
    defaultLanguage: 'en',
    languages: {
      en: { name: 'English', path: 'NTEnglish/Audio', format: 'PostXXEnglish.mp3' },
      fr: { name: 'Français', path: 'NTFrench/Audio', format: 'PostXXFrench.mp3' },
      hi: { name: 'हिन्दी', path: 'NTHindi/Audio', format: 'PostXXHindi.mp3' },
      pl: { name: 'Polski', path: 'NTPolish/Audio', format: 'PostXXPolish.mp3' }
    },
    stopTitles: {
      1: 'Introduction',
      2: 'The Viewpoint',
      // ... etc
    }
  });
</script>

Gallery Lightbox Component

Click any image to open the lightbox viewer

Features:

  • Click to enlarge images
  • Previous/next navigation
  • Image counter
  • Captions support
  • Keyboard navigation (Esc to close, Arrow keys to navigate)
  • Close on overlay click
  • Mobile responsive
  • Loading spinner
  • Fully accessible with ARIA labels

Usage

<!-- Add to your HTML -->
<div class="gallery">
  <figure class="gallery-item" data-lightbox>
    <img src="path/to/image.jpg" alt="Description">
    <figcaption>Optional caption</figcaption>
  </figure>
  <!-- More gallery items... -->
</div>

<!-- Include CSS and JS -->
<link rel="stylesheet" href="css/style.css">
<script src="js/components.js"></script>

<!-- Components auto-initialize, or initialize manually: -->
<script>
  const gallery = new GalleryLightbox('[data-lightbox]', {
    animationDuration: 300,
    closeOnOverlayClick: true,
    showCounter: true,
    keyboardNav: true
  });
</script>

Mobile Menu Component

Enhanced mobile navigation (resize browser to see)

Features:

  • Hamburger menu toggle
  • Close on link click
  • Close on outside click
  • Keyboard support (Esc to close)
  • Focus management for accessibility
  • Automatic initialization

Usage

<!-- Add to your HTML -->
<header>
  <nav>
    <button class="menu-toggle" aria-label="Toggle menu">☰</button>
    <ul class="nav-links">
      <li><a href="#">Home</a></li>
      <li><a href="#">About</a></li>
      <!-- More links... -->
    </ul>
  </nav>
</header>

<!-- Include CSS and JS -->
<link rel="stylesheet" href="css/style.css">
<script src="js/components.js"></script>

<!-- Automatically initializes, or manually: -->
<script>
  const menu = new MobileMenu('.menu-toggle', '.nav-links', {
    activeClass: 'active',
    closeOnLinkClick: true,
    closeOnOutsideClick: true
  });
</script>

Technical Notes

Browser Support

All modern browsers including:

  • Chrome/Edge 90+
  • Firefox 88+
  • Safari 14+
  • Mobile browsers

Progressive Enhancement

Components are designed to fail gracefully:

  • Images work without JS
  • Native audio controls fallback
  • Menu accessible via CSS

Accessibility

Built with a11y in mind:

  • ARIA labels and roles
  • Keyboard navigation
  • Focus management
  • Screen reader support

File Structure

site-2026/
├── css/
│   └── style.css          (includes component styles)
├── js/
│   └── components.js      (all components)
├── index.html
└── component-examples.html (this file)

No Dependencies

All components are built with vanilla JavaScript - no jQuery, React, or other frameworks required. This keeps the site fast, lightweight, and easy to maintain.
